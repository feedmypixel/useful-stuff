# Useful Mac Stuff

## Screen shot
#### CrossHair
```
cmd + shift + 4 
```

#### Capture screen
```
cmd + shift + 3
```

#### Open finder from command line
```
open .
```


#### Diff files from cmd line on mac
```
opendiff <filename_1> <filename_2>
```

#### Run program in background
```
program name &
```

#### Send job to foreground
```
fg <pid>
```
