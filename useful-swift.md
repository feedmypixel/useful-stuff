# Useful Swift Stuff

#### Init project
```
$ swift package init --type executable
```

#### Generate Xcode Project
```
$ swift package generate-xcodeproj
```

#### REPL
Open the swift REPL on the command line

```
$ swift
```

#### Current version
```
$ swift -version
```

#### Useful commands

$ swift package
